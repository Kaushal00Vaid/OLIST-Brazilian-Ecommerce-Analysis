# OLIST Brazilian E-commerce Dataset Analysis

## Dataset Link
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## Analysis

### Theme A -- Understanding the business

1. A1 - Revenue & order trends over time
   - Monthly GMV, order volume, avg order value.
2. A2 - Category performance breakdown
   - Revenue, bolume, avg price by category. Which categories drive the business vs which are noise.

### Theme B -- Where Operations Fail

1. B1 - Delivery performance deep-dive
   - Late rate by state, avg delay in days, worst performing regions.
2. B2 - Seller performance ranking
   - Revenue, late rate, avg review score per seller. Identifies who's dragging down the platform.

### Theme C -- Customer Behaviour

1. C1 - Review score distribution & drivers
   - Score breakdown, avg score by delivery status (late vs on-time). Connects ops failure to customer feeling.
2. C2 - Cohort retention analysis
   - Monthly cohorts by first purchase. What % return month 1, 2, 3? Shows longitudinal thinking.
